# The Blue Heron Black Hills

Production-ready Next.js hospitality website starter.

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

Recommended hosting:
- Vercel
- Domain via Porkbun

## Features
- Mobile responsive
- SEO optimized
- Cinematic dark aesthetic
- Framer Motion transitions
- Blog/journal structure
- Event pages
- Square integration placeholders
